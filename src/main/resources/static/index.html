⚡ CMS API Reference

Complaint Management System · Spring Boot REST API

Base URL: http://localhost:8080/api/complaints

📋 Core CRUD

GET/List all complaints (paginated)
POST/Create a new complaint
GET/{id}Get complaint by ID
PUT/{id}Update complaint details
DELETE/{id}Delete a complaint

🔍 Filtering

GET/status/{status}OPEN · IN_PROGRESS · RESOLVED · CLOSED · REJECTED
GET/category/{category}PRODUCT · SERVICE · BILLING · TECHNICAL · OTHER
GET/priority/{priority}LOW · MEDIUM · HIGH · CRITICAL
GET/email/{email}All complaints by email
GET/search?keyword=Full-text search (title, description, name)

⚡ Patch Actions

PATCH/{id}/status?status=&resolutionNote=Update status
PATCH/{id}/priority?priority=Update priority

📊 Analytics

GET/statsDashboard statistics
GET/urgentHIGH + CRITICAL open complaints
GET/recent?limit=5Most recent N complaints
GET/date-range?start=&end=Filter by date range (ISO 8601)

🔧 Dev Tools

GET/h2-consoleH2 Database Console (dev only)
GET/actuator/healthHealth check
